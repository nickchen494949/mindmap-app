# 虚拟大脑 — 产品愿景

> 记录日期：2026-06-03
> 状态：愿景规划阶段

---

## 核心理念

**工具是开源的，大脑是私有的。**

把 deep research 的成果 + 用户自己的想法 + 实时事件，连接成一棵不断生长的知识树。这棵树就是用户的虚拟大脑。然后从大脑里自动生成视频、图片、文案，变成自媒体引擎。

---

## 三层产品架构

### 第一层：虚拟大脑（知识图谱）

```
Deep Research → AI 生成知识骨架（mindmap）
User 剪切想法 → 挂到节点上（主观判断层）
新事件出现 → AI 自动找到该挂在哪 → 大脑自动生长
```

**关键设计：**
- 每个节点有 `source` 字段区分来源：
  - `research` — AI 研究出来的客观知识
  - `user` — 用户自己的想法、判断、经验
  - `event` — 新闻/事件，自动或手动挂进来
- 节点之间的箭头 = 因果关系 / 逻辑连接
- 时间越久，图越密，认知越深，护城河越深
- 别人能抄工具，抄不了你的大脑

**为什么有壁垒：**
- 你的节点里有你自己的判断
- 你的连线代表你独特的因果认知
- 积累越久，输出越独特

---

### 第二层：自动出片（内容生成引擎）

```
mindmap JSON
  → 自动生成演示视频 + 字幕 + AI 语音旁白
  → 自动切图做 infographic
  → 自动写文案（小红书 / Twitter / 公众号）
```

**已有基础：**
- ✅ 演示模式（自动缩放 + 高亮 + 字幕）
- ✅ SRT 字幕导出
- 🔲 片头/片尾卡
- 🔲 Bloomberg 式 lower-third 字幕
- 🔲 浏览器 TTS 语音旁白
- 🔲 直接导出 MP4
- 🔲 自动生成 infographic 图片

---

### 第三层：自媒体引擎（自动发布）

```
每周新事件
  → 挂到你的大脑树
  → 自动生成"本周解读"视频
  → 自动发布到各平台
```

**长期目标：**
- 定时抓取新闻/事件（RSS / API）
- AI 判断该挂到大脑的哪个节点
- 自动生成解读内容
- 一键发布到多个平台
